# Mini-Shell

En este reto, implementé una mini-shell en Node.js. La mini-shell es un programa que simula el comportamiento de una terminal, permitiendo al usuario ejecutar comandos y ver sus resultados.

# Características

- Soporta comandos para navegar por el sistema de archivos (cd, ls).
- Escanea el PATH para encontrar ejecutables.
- Permite ejecutar comandos externos.
- Maneja errores de comandos no encontrados o rutas inválidas.

# Nota

Debido al tiempo limitado de 90 minutos, la implementación se centró en lograr una funcionalidad básica y no en optimizar el código o manejar casos edge. Los comandos están hardcodeados para simplificar la lógica y asegurar que se puedan ejecutar dentro del tiempo asignado.
